# repos Directory

This directory contains Spack package recipes and patches that have been customized, extended, or created specifically for the RIS Compute2 environment.

## Structure

- **ris/**  
  Custom and modified Spack package files for RIS.

## Usage

Add this repository to your Spack configuration to use these custom packages:

```bash
spack repo add /path/to/compute2-spack/repos/ris
```

Refer to the main project README for deployment and usage

# compute2-spack-packages
Primary RIS Spack package repository


## Summary
This repository is designed to be consumed directly via URL by Spack v1.0+ as a remote [Git-based Repository](https://spack.readthedocs.io/en/latest/repositories.html#git-based-repositories).

It is also setup as a [Git Submodule](https://git-scm.com/book/en/v2/Git-Tools-Submodules) within the [Compute2 Spack Repo](https://githhub.com/washu-it-ris/compute2-spack) to enable "normal" deployment per instructions there.

## Quick-Start

### Setup Remote Repo

1. Add the repository from the CLI
   ```bash
   spack repo add https://github.com/washu-it-ris/compute2-spack-packages.git
   ```
